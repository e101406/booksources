[http://ku.mumuceo.com](http://ku.mumuceo.com)

### 说明
每隔一段时间书源就大批量失效，为了方便阅读APP书源更新使用，特地整合了这个书源版本，后续会增加其它版本 

####目录约定
```$xslt
- all（所有书源）
- 2.0（适用2.0版本书源）
- 3.0（适用3.0版本书源）
        // 以上每一个目录下都拥有有（不存在就是没有源）
        --- total.json（全部书源）
        --- is_audio.json（书源支持音频）
        --- is_search.json（书源支持发现）
        --- is_discover.json（书源支持搜索）
```
